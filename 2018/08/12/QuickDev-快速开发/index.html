QuickDev(快速开发)

后端 开发环境 2018/08/12


QuickDev是一个集成Java web后端开发环境的工具包，目标是帮助Java后端开发人员，快速搭建开发环境，节约开发环境创建时间，快速进入开发状态。

集成软件

1. JDK "1.8.0_131"
2. Apache Maven 3.3.3
3. Git version 2.18.0.windows.1
4. IntelliJ IDEA 2018.2 (Ultimate Edition)
5. MariaDB 10.3.8
6. Tomcat 7.0
7. Tomcat 8.0

目录结构

D:.                       // 盘符
├─dev                    // 主目录
│  ├─bin                // 脚本及其他
│  ├─opt               // 软件环境
│  │  ├─Git
│  │  ├─Java
│  │  ├─JetBrains
│  │  ├─mariaDB
│  │  ├─Maven
│  │  ├─Tomcat7.0
│  │  └─Tomcat8.5
│  └─var             // 工作空间
│      ├─git
│      ├─m2
│      │  └─repo
│      ├─workspace_idea

详细说明

1. Maven 仓库默认配置到阿里云的仓库
2. Maven 仓库位置默认位置为 var/m2/repo
3. mariaDB 默认用户名密码为 root/root
4. Tomcat7.0和Tomcat8.5
5. bin目录存在两个reg文件，主要作用是将管理员打开命令行加入右键菜单
   cmd.reg 注册服务到注册表
   rmcmd.reg 从注册表移除

IntelliJ IDEA 2018.2 说明

IntelliJ IDEA 2018.2 (Ultimate Edition) 旗舰版，注册日期到2019-05-05

  1. 默认配置文件路径已修改至安装目录D:\dev\opt\JetBrains\IntelliJ IDEA 2018.2.IntelliJIdea
  2. 如果出现激活失败，请按照如下方法激活：

    软件注册

    1. 修改hosts文件，win10系统，路径如下：
      C:\Windows\System32\drivers\etc\hosts
      在文件中添加解析地址，内容如下：
      0.0.0.0 account.jetbrains.com
    2. 填写激活码，激活码文件在D:\dev\opt\JetBrains下的IntelliJ IDEA CODE.txt,将内容全部复制，然后打开IDEA,粘贴到激活码输入框即可。

关于代码格式化

  1. 默认使用Eclipse Code Formatter,已经在目录里面提供了一份格式化的模板，需要修改请自行调整，文件路径：
    D:\dev\opt\JetBrains\IntelliJ IDEA 2018.2.IntelliJIdea\Eclipse [customer].xml
  2. 保存代码自动格式化，优化导包，自动添加实现方法…默认已设置，如果失效，请检查Save Actions设置
  3. 默认已做了很多方面设置，如字体类型，字体大小，代码注释等等，请按需要自行调整
